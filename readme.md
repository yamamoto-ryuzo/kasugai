## ライセンスについて

本システム（Kasugai）は、**MITライセンス**の下で公開されています。詳細については [LICENSE](./LICENSE) ファイルを参照してください。

本システムが依存している主要なサードパーティ製ライブラリおよびツールの一覧、およびそのライセンス情報は以下の通りです。いずれもMITライセンスと互換性があり、MITライセンスでのシステム公開において問題ありません。

### バックエンド（Rust / Cargo 依存関係）
| ライブラリ / クレート名 | ライセンス | 用途 |
| :--- | :--- | :--- |
| **tauri (v2)** | MIT または Apache-2.0 | デスクトップアプリケーションフレームワーク |
| **serde** | MIT または Apache-2.0 | シリアライズ/デシリアライズ |
| **serde_json** | MIT または Apache-2.0 | JSONデータの取り扱い |
| **keyring (v2)** | MIT または Apache-2.0 | OS資格情報ストア連携 |
| **enigo (v0.1.2)** | MIT | キーボード・マウスエミュレーション |
| **tauri-build (v2)** | MIT または Apache-2.0 | ビルドスクリプト支援 |

### フロントエンド（Web / CDN 依存関係）
| ライブラリ名 | ライセンス | 用途 |

| **CesiumJS** | Apache-2.0 | 3D地球儀プラットフォーム・Web GIS |
