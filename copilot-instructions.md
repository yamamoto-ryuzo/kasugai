# Kasugai プロジェクトのビルド方法

このプロジェクトは Tauri を使用しています。以下の手順でビルドしてください。

1.  **プロジェクトのルートディレクトリに移動します。**
    このワークスペースを開いている場合は、すでに正しい場所にいるはずです。

2.  **`kasugai` サブディレクトリに移動します。**
    ```bash
    cd kasugai
    ```

3.  **依存関係をインストールします。**
    ```bash
    npm install
    ```

4.  **アプリケーションをビルドします。**
    ```bash
    npm run tauri build
    ```

5.  **ビルド成果物を確認します。**
    ビルドが完了すると、実行可能ファイルとインストーラーが以下のディレクトリに生成されます。
    -   `src-tauri/target/release/`
    -   `src-tauri/target/release/bundle/`
